24  '''A logger for GNUWiNetwork modules.

26  This module provides configuration for the logging facilities used in the GNUWiNetwork modules.

28  See:  See I{"Logging", "Logging HowTo", and "Logging Cookbook" in Python Documentation}.

30  @var appname: application name, default 'GNlogger'. 
31  @var fname: log file name, default 'gnlogger.log'. 
32  @var fmode: log file mode, default 'w'. Mode 'a' appends to existing file, mode 'w' starts a new log file. 
33  @var loglvl: less sever log level to start logging, default DEBUG. In increasing order of severity: DEBUG, INFO, WARNING, ERROR, CRITICAL. Default C{logging.DEBUG}. See I{Logging HowTo} in Python Documentation. 
34  @var fmtstr: format string to define format of log output. 
35  @var logdir: directory to place log files, defaults to C{../logs}. 
36  '''
